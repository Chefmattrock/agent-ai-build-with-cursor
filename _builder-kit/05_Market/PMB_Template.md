**AEO (Answer Engine Optimization) Strategy:**

**1. Query Fan-out Analysis**
*Ensure H2s cover multiple synthetic query types to maximize AI-driven search visibility*

Check if your content addresses these query types:

- **Related Queries:** [Semantically adjacent topics]
  - Example: "top rated [category]", "best [related product]"
- **Implicit Queries:** [Inferred user intent]
  - Example: "[Feature] with longest [attribute]", "affordable [use case]"
- **Comparative Queries:** [Decision-making comparisons]
  - Example: "[Product A] vs. [Product B]", "[Category] comparison chart 2025"
- **Entity-Expanded Queries:** [Broader/narrower entity relationships]
  - Example: "[Specific model] reviews", "[Brand A] vs [Brand B]"
- **Reformulation Queries:** [Alternative phrasings of core intent]
  - Example: "which [product] is the best", "top [category] for [year]"

**Goal:** Cover 3-5 different query types across your H2 structure to maximize LLM retrieval.

**For Product Pages:**
